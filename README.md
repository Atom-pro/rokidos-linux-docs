## RokidOS 开放源代码项目简介
RokidOS 是 Rokid 领导的基于云端语音服务平台的一个可以广泛应用于智能音箱、车载系统、智能电视、机顶盒等多种设备的开放源码代码语音交互系统。您可以用 Rokid 提供的技术，为您的客户提供自然的语音交互体验。我们整合了大量的工具、APIs、参考解决方案以及文档，让任何人都能轻松打造具备语音交互能力的设备。
